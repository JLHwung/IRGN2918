# IRGN2918

The sources of IRGN2918. See [Releases](https://github.com/JLHwung/IRGN2918/releases) for the generated PDF.

## Build

Requirements:

- [TeX Live](https://www.tug.org/texlive/)
- [GNU Make](https://www.gnu.org/software/make/)

```sh
python -m venv .venv
.venv/bin/pip install -r ./requirements.txt
make
```
